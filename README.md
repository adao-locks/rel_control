## How to Run the Project

First you need to run:

`flutter pub get`

To check status run:

`flutter doctor`

Then finally:

`flutter run`

### Maybe necessaary

You may need to update flutter, you can run:

`flutter upgrade`

Or, you can force the update

`flutter channel stable` <br>
`flutter upgrade`

After all you need to run `flutter pub get` again.
